Connexus is a photo sharing web application.
Users can:
- Create image streams
- Subscribe to other users' image streams
- Upload images to their stream using multi-upload feature
- View the most popular and trending stream
- Set email notifications for most popular and trending streams
- Search for any image or stream by (date, tag, name, user, subscriber)
- View images geographically on a map

Connexus was built as a project for the "Advanced Programming Tools" course at The University of Texas at Austin. The team members that worked on the project are:
- Elie Antoun
- Erik Schilling

The backend was built using Google App Engine (python)(required by the course and by the course instructors).
